## 📖 Overview

The **Loan Application Mathematics & Statistics** project provides an in-depth analysis of loan application data, focusing on statistical methods to assess loan default risks. Utilizing Python's data science stack, this project offers insights into factors influencing loan approvals and rejections.

---

## ✨ Features

- 📊 **Exploratory Data Analysis (EDA):** Visualizes distributions of applicant demographics and loan statuses.
- 🔄 **Data Preprocessing:** Handles missing values, encodes categorical variables, and scales numerical features.
- 🤖 **Model Training:** Implements machine learning models to predict loan approval probabilities.
- 📈 **Model Evaluation:** Assesses model performance using accuracy, precision, recall, and F1-score metrics.
- 📄 **Reporting:** Generates comprehensive reports on loan default risks and approval factors.

---

## 🧠 Concepts Covered

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)

## 🚀 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/patelneel9080/Loan_Application_Mathematics_Statistics.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Loan_Application_Mathematics_Statistics
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

---

## 💡 Usage

1. Open the Jupyter Notebook:

   ```bash
   jupyter notebook Loan_Application.ipynb
   ```

2. Execute the cells sequentially to perform data analysis and model training.

## 📈 Key Insights

- **Applicant Income:** Higher applicant income correlates with increased loan approval rates.
- **Credit History:** Applicants with a positive credit history (1.0) have a higher chance of loan approval.
- **Property Area:** Urban areas show a higher loan approval rate compared to semiurban and rural areas.
- **Education Level:** Graduates are more likely to have their loan applications approved.
- **Marital Status:** Married applicants tend to have a higher loan approval rate than single applicants.
